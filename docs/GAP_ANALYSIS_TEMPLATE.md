# Gap Analysis Template

Use this document to record your prioritized learning plan after running the skill assessment. Fill it in after running `python scripts/analyze_gaps.py` and reviewing the output.

## How to use this template

1. Run `python scripts/analyze_gaps.py` and copy your top gaps into the tables below.
2. For each gap, choose the weeks from the template you will use to close it.
3. Set a target completion date and note any blockers or resources needed.
4. Revisit this document at mid-program (Week 5) and end-of-program (Week 10) to update progress.

- **Medium** — Gap of 2, or domain is important but not blocking
- **Low** — Gap of 1, or domain is a stretch goal beyond current role scope
